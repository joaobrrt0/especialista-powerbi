# 📊 Dashboard de Vendas em Power BI

Este projeto apresenta um **Dashboard interativo de Vendas** desenvolvido no Power BI, com o objetivo de analisar o desempenho de vendedores ao longo do ano de 2025.

## 🗂️ Visão Geral

O relatório é dividido em duas páginas principais:

### 1. **Painel de Análise Geral**
- **Filtro de Período de Venda**
- **Filtro por Vendedor**
- **Indicadores Principais (KPIs):**
  - Total de Receitas
  - Receita no Período
  - Percentual do Total de Receita
- **Gráficos:**
  - Receita por Vendedor
  - Receita por Produto (por Vendedor)
  - Evolução Mensal da Receita Total e Média

### 2. **Tabela Detalhada de Vendas**
- Tabela com os dados de vendas por data e vendedor
- KPIs específicos como:
  - Média de Vendas (geral e por vendedor)
  - Soma de Valor Total Travada (referência fixa)
  - Filtro de intervalo de datas para análise específica

## 📁 Dados Utilizados

Os dados utilizados neste relatório incluem:
- **Data da Venda**
- **Nome do Vendedor**
- **Produto Vendido**
- **Valor Total da Venda**

*Observação: os dados foram simulados para fins de visualização.*
